# minesweeper
This program is console version of minesweeper.  
Game has three mods: a, b, c.  
a: size of the field is 5 and number of bombs is 2-5  
b: size of the field and number of the bombs are set by the user  
c: the program generates and solves the game
Symbols on the field:  
'-' means closed cell  
'*' means bomb  
number represents the amount of the bombs in the neighbor cells
